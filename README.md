## Confidencialidad profesional

Parte de mi experiencia se ha desarrollado en proyectos reales para empresas y clientes.  
Por responsabilidad profesional, no publico:

- Código de clientes.
- Credenciales.
- URLs internas.
- Endpoints privados.
- Datos comerciales.
- Reservas, usuarios o datos reales.
- Capturas sensibles.
- Evidencias de ambientes privados.

Cuando se menciona experiencia de cliente, se hace solo a nivel general, técnico y profesional.
